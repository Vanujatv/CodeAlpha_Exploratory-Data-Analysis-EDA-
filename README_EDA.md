# CodeAlpha - Exploratory Data Analysis (EDA)

## Overview
This project is part of the CodeAlpha Data Analytics Internship. It performs Exploratory Data Analysis (EDA) on a Yearly Sales dataset to understand the data, identify patterns, detect missing values, and generate insights.

## Dataset
- Yearly Sales Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## EDA Steps
- Data Loading
- Data Inspection
- Missing Value Check
- Duplicate Check
- Summary Statistics
- Data Visualization
- Insights

## Conclusion
The dataset was explored successfully, and meaningful insights were obtained through statistical analysis and visualization.
